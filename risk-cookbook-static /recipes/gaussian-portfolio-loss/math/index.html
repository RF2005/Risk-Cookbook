Recipe 01 - Math

Gaussian Portfolio Loss

The mathematics behind the Gaussian portfolio loss recipe.

Back to recipe

Model setup

The recipe starts with a return vector \(R \in \mathbb{R}^n\) over one horizon and assumes

\[ R \sim \mathcal{N}(\mu, \Sigma) \]

where \(\mu\) is the mean return vector and \(\Sigma\) is the covariance matrix. In the current page, \(\Sigma\) is built from user-entered volatilities and one common correlation \(\rho\).

\[ \Sigma_{ij} = \sigma_i \sigma_j \rho_{ij}, \qquad \rho_{ij} = \begin{cases} 1 & i=j \\ \rho & i \ne j \end{cases} \]

From returns to portfolio loss

If \(w\) is the vector of portfolio weights, then portfolio return is \(w^\top R\). The page works with loss, so it flips the sign:

\[ L = -w^\top R \]

This makes larger values of \(L\) correspond to worse portfolio outcomes.

Why the loss is Gaussian

A linear combination of jointly Gaussian random variables is still Gaussian. Since \(L\) is a linear map of \(R\), its distribution is

\[ L \sim \mathcal{N}\left(-w^\top \mu,\; w^\top \Sigma w\right) \]

So the page is really computing two objects:

\[ \mathbb{E}[L] = -w^\top \mu, \qquad \mathrm{Var}(L) = w^\top \Sigma w \]

Tail metrics

For a confidence level \(\alpha\), the page reports Value at Risk and Expected Shortfall.

\[ \mathrm{VaR}_{\alpha}(L) = F_L^{-1}(\alpha) \]
\[ \mathrm{ES}_{\alpha}(L) = \mathbb{E}[L \mid L \ge \mathrm{VaR}_{\alpha}(L)] \]

Because \(L\) is Gaussian, there are closed-form expressions in terms of the standard normal quantile \(z_\alpha\):

\[ \mathrm{VaR}_{\alpha}(L) = -w^\top \mu + \sqrt{w^\top \Sigma w}\; z_\alpha \]
\[ \mathrm{ES}_{\alpha}(L) = -w^\top \mu + \sqrt{w^\top \Sigma w}\; \frac{\varphi(z_\alpha)}{1-\alpha} \]

where \(\varphi\) is the standard normal density.

What the simulation is doing

The simulation on the main recipe page generates many independent draws \(R^{(1)}, \dots, R^{(M)}\) from \(\mathcal{N}(\mu, \Sigma)\). Each draw is converted to a loss:

\[ L^{(m)} = -w^\top R^{(m)} \]

The histogram is therefore a Monte Carlo approximation of the Gaussian loss distribution. The path panel uses repeated Gaussian draws over several steps to show how portfolio value can evolve through time under the same one-step assumption.

This page is the baseline recipe. The later pages change the return law, the data source, or the covariance structure, but they still map everything back into the same core object: portfolio loss.
