Recipe 01

Gaussian Portfolio Loss

A baseline model for portfolio loss built from weights, means, volatilities, and correlation.

Math behind this recipe

Inputs

Enter one value per asset.
Use horizon-level returns.
Use horizon-level standard deviations.

Intuition

Start with a vector of asset returns, assume the returns are jointly Gaussian, then map those returns into a portfolio loss with a weight vector.

General recipe

  1. Choose the number of assets and the portfolio weights.
  2. Enter mean returns, volatilities, and a common correlation.
  3. Build a covariance matrix from the volatilities and correlation.
  4. Simulate returns and convert them into portfolio losses.
  5. Compute mean loss, loss volatility, VaR, and expected shortfall.
