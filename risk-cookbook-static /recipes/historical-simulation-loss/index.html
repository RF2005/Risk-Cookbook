Recipe 03

Historical Simulation Loss

An empirical loss distribution built directly from observed return scenarios.

Math behind this recipe

Inputs

Paste CSV with a date column followed by one return column per asset.

Intuition

Instead of assuming a return distribution, historical simulation replays observed historical scenarios on the current portfolio and reads off the empirical tail.

General recipe

  1. Paste a panel of historical returns.
  2. Apply your current portfolio weights to every historical row.
  3. Sort the resulting loss scenarios.
  4. Read off historical VaR and expected shortfall.
