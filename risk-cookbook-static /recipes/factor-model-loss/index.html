Recipe 04

Factor Model Loss

A structured risk model using common factors and specific risk.

Math behind this recipe

Inputs

One row per asset, one column per factor.

Intuition

Factor models explain asset returns through a smaller number of common shocks plus idiosyncratic noise. That makes large portfolios easier to interpret and decompose.

General recipe

  1. Enter your portfolio weights.
  2. Enter factor means, volatilities, and common factor correlation.
  3. Enter the exposure matrix linking assets to factors.
  4. Add specific risk for each asset.
  5. Simulate factor returns and specific shocks, then convert them into portfolio loss.
