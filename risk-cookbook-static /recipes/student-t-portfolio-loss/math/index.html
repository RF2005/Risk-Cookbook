Recipe 02 - Math

Student-t Portfolio Loss

The mathematics behind the Student-t portfolio loss recipe.

Back to recipe

Model setup

This page keeps the same loss map as the Gaussian recipe but changes the return law to a heavy-tailed Student-t model:

\[ R \sim t_\nu(\mu, \Sigma) \]

Here \(\nu\) is the degrees-of-freedom parameter. Lower values of \(\nu\) create fatter tails. As \(\nu \to \infty\), the Student-t distribution approaches a Gaussian law.

From returns to loss

Just as before, the page defines loss as

\[ L = -w^\top R \]

The main conceptual point is that the mapping from returns to loss does not change. Only the tail behavior of the returns changes.

Why the tails are heavier

A multivariate Student-t variable can be written as a Gaussian scale mixture:

\[ R = \mu + \frac{AZ}{\sqrt{U/\nu}} \]

where \(Z \sim \mathcal{N}(0, I)\), \(U \sim \chi^2_\nu\), and \(AA^\top = \Sigma\). The random denominator makes some draws much larger in magnitude than the Gaussian case, which is exactly why the tail metrics increase.

Loss distribution and tail metrics

Because \(L\) is still a linear combination of the return vector, its distribution is also Student-t with the same degrees of freedom and an induced scale based on \(w\) and \(\Sigma\):

\[ L \sim t_\nu\bigl(-w^\top \mu,\; w^\top \Sigma w\bigr) \]

The page estimates

\[ \mathrm{VaR}_{\alpha}(L), \qquad \mathrm{ES}_{\alpha}(L) \]

by Monte Carlo simulation rather than closed form, then compares those values with a Gaussian benchmark using the same mean and covariance inputs.

What the simulation is doing

The main recipe generates many Student-t return vectors, converts them into losses, and uses those simulated losses to approximate the tail of \(L\). The path panel compounds a sequence of Student-t shocks to show how rare but severe drawdowns become more likely than under the Gaussian model.

This page is useful because it isolates one question: what changes if the portfolio has the same broad covariance structure, but the shocks are heavy-tailed instead of Gaussian?
